## Dependencies  
> [Jekyll](https://jekyllrb.com/)  
`gem install jekyll bundler`  

## Local Env
> Build & serve
`bundle exec jekyll serve`

## Git  
> Get changes from Github into local environment
`git pull`

> Push local changes to Github
`git push origin master`
